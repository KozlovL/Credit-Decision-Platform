# Contributing

Спасибо, что хотите внести вклад в проект!

## Как начать
1. Сделайте форк репозитория
2. Создайте новую ветку для своей задачи:
   ```bash
   git checkout -b feature/my-feature
   ```
3. Внесите изменения и сделайте коммит:
   ```bash
   git commit -m "Добавлена новая фича"
   ```
4. Отправьте изменения в свой форк:
   ```bash
   git push origin feature/my-feature
   ```
5. Откройте Pull Request в основной репозиторий.

## Правила
- Соблюдайте стиль кода (PEP8 для Python)
- Перед коммитом прогоняйте тесты:
  ```bash
  pytest
  ```
- Обновляйте `CHANGELOG.md` для каждого значимого изменения
- Используйте говорящие сообщения коммитов

## Контакты
Если есть вопросы — создайте issue или напишите в командный чат.
